# Disaster Management Alert System (DMAS)

A modular Tkinter-based desktop application for simulating and managing disaster alerts.

## Structure
- `gui/main_gui.py` - main GUI interface
- `backend/alert_logic.py` - disaster detection and alert logging
- `data/alerts_log.txt` - stores logs of alerts

## Usage

- Run `python gui/main_gui.py`
- Click disaster buttons to trigger and log alerts

## Future Features

- Integrate with sensors or remote APIs
- Add real-time data visualization
- Implement notification via email or SMS
